
Array is a data type structure that stores many pieces of data in the same variable. As a best practise, an array should store data pieces that are the same type (all strings / all numbers, etc). 


## Defining an array

As shown below, arrays can be defined in two different ways:

```
<?php
	// An old-fashioned legitimate way of defining an array
	$fruits_1 = array("Apple", "Banana", "Cherry");

	// A shorter syntax, available from PHP version 8 or above
	$fruits_2 = ["Apple", "Banana", "Cherry"];
?>
```

Since PHP version 8, it is allowed to leave a comma sign  ` , `  after the last element in the array, and it will not cause any error to be thrown :

```
<?php
	// Trailing comma is syntactically correct since PHP version 8 or above
	$fruits_with trailing_comma = ["Apple", "Banana", "Cherry",]
?>
```


## Accessing the array

In order to access a specifc element of an array, a **square bracket notation** can be used. Inside the square brackets you should put a number that represents an index of element to be retrieved. The index is zero-based, meaning that to get the first element of the array, the corresponding index will be 0 :

```
<?php
	$fruits = [
		"Apple",    // index: 0
		"Banana",   // index: 1
		"Cherry"    // index: 2
	];
	
	echo $fruits[0];  // Output: "Banana"
	echo $fruits[2];  // Output: "Cherry"
?>
```


## Adding an element to array

PHP contains a special syntax to add a new element to an array. In order to insert a new element at the end of the array, just **leave the square brackets empty** (do not specify an index) :

```
<?php
	$fruits = ["Apple", "Banana", "Cherry"];

	$fruits[] = "Mango";
	$fruits[] = "Orange";
	
	echo $fruits[3];  // Output: "Mango"
	echo $fruits[4];  // Output: "Orange"
?>
```


## Modifying existing array element

To modify an already present array element, you need to **target existing index inside the square brackets**, and then assign this entry to a new value, like so :

```
	$fruits = ["Apple", "Banana", "Cherry"];

	$fruits[1] = "Watermelon";

	echo $fruits[1];  // Output: "Watermelon"
```
