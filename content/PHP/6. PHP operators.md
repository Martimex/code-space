Every programming language contains a special set of characters that are specifcally used when performing a given task. That could be: math operations, assigning variables, comparing numbers and much more!

Below is the list of the most important ones, grouped into use-case categories. Each operator will include at least one code example to show them in action.

## String operators

### Dot operator ( `.` )

The dot operator is used to concatenate strings. It makes possible to compose a longer string from two (or more!) smaller ones.

```
<?php
	$word1 = "Hello";

	echo $word1 . "World!";   // Output: "HelloWorld!"
	echo $word1 . " " . "World!";   // Output: "Hello World!"
?>
```

It is important to keep a space between the `dot operator` and the variables (or strings) to be concatenated. Therefore, this code example will NOT work:

```
<?php
	$word1 = "Good";

	echo $word1."Morning"; // NOT WORKING - DOT OPERATOR REQUIRE SPACES IN-BETWEEN
?>
```


## Arithmetic operator

### Addition operator ( `+` )

The addition operator is used to add two numbers, resulting in a sum of the two values.

```
<?php
	$value1 = 10;
	$value2 = 33;

	echo $value1 + $value2;   // Output: 43
	echo $value2 + 31;   // Output: 64
	echo $value1 + $value1;   // Output: 20
	echo 7 + 3;  // Output: 10
?>
```

### Substraction operator ( `-` )

The substraction operator is used to substract one number from another, resulting in numerical difference between them:

```
<?php
	$value1 = 20;
	$value2 = 15;

	echo $value1 - $value2;  // Output: 5
	echo $value1 - 2;   // Output: 18
	echo $value2 - $value2;   // Output: 0
	echo 88 - 11;   // Output: 77
	echo 5 - 15;   // Output: -10
?>
```

### Multiplication operator ( `*` )

The multiplication operator is used to multiply a given number by the amount of another number:

```
<?php
	$value1 = 12;
	$value2 = 5;

	echo $value1 * $value2;  // Output: 60
	echo $value1 * 10;   // Output: 120
	echo $value2 * value2;   // Output: 25
	echo 24 * 3;   // Output: 72
?>
```

### Division operator ( `/` )

The division operator is used to dive a number by another number :

```
<?php
	$value1 = 90;
	$value2 = 3;

	echo $value1 / $value2;   // Output: 30
	echo $value1 / 2;   // Output: 45
	echo $value2 / $value2;  // Output: 1
	echo 120 / 6;   // Output: 20
?>
```

### Modulo operator ( `%` )

The modulo operator returns the remainder value of the operation. When left-hand side value cannot contain more of added right-hand side values in itself, then the remainder is the remaining amount left (always not enough) to include the next right-hand side value inside the left-hand side value.

```
<?php
	$value1 = 10;
	$value2 = 3;

	echo $value1 % $value2;   // Output: 1 (10 - (3 + 3 + 3))
	echo $value2 % $value1;   // Output: 3 (left-hs < right-hs; returns left-hs)
	echo $value2 % value2;    // Output: 0 (3 - 3)
	echo $value1 % 4;   // Output: 2 (10 - (4 + 4))
	echo 17 % 6;   // Output: 5 (17 - (6 + 6))
?>
```

### Power-of operator ( `**` )

The power-of operator sets the left-hand side operator to the value of left-hand side operator multiplied by right-hand side amount of times.

```
<?php

	$value1 = 10;
	$value2 = 4;

	echo $value1 ** $value2;   // Output: 10000
	echo $value2 ** $value1;   // Output: 1048576
	echo $value2 ** 3;   // Output: 64
	echo $value1 ** $value1;   // Output: 10000000000
	echo 5 ** 4;   // Output: 625
?>
```

### Precedence operator (   `( )`  )

Precedence operator can be used to change the default calculation order in mathematics.

```
<?php
	// Without the precedence operator, multiply is calculated before addition
	echo 2 + 10 * 5;   // Output: 52

	// After putting addition operation inside precedence operator, it will be              // calculated before multiplying:
	echo (2 + 10) * 5;  // Output: 60
?>
```


## Assignment operators

### Pure assignment operator ( `=` )

A standard assignment operatur used to assign variables to specifc values. It also can be used to reassign an existing variable to a new value:

```
<?php
	$pao = 6; 
	echo pa;   // Output: 6
	$pao = 24;
	echo pa;   // Output: 24
?>
```

### Assignment addition operator ( `+=` )

**Operator used only for reassigning values**. It reassigns the variable to its' existing value, but also adds the right-hand side value to the final left-hand side variable value:

```
<?php
	$aao = 12;

	$aao += 20;
	echo aao;   // Output 32
?>
```

### Assignment substraction operator ( `-=` )

**Operator used only for reassigning values**. It reassigns the variable to its' existing value, but also substracts the right-hand side value from the final left-hand side variable value:

```
<?php
	$aso = 120;

	$aso -= 10;
	echo aso;   // Output 110
?>
```

### Assignment multiplication operator ( `*=` )

**Operator used only for reassigning values**. It reassigns the variable to its' existing value, but also multiplies left-hand side variable value by right-hand side value:

```
<?php
	$amo = 15;

	$amo *= 6;
	echo amo;   // Output 90
?>
```

### Assignment division operator ( `/=`)

**Operator used only for reassigning values**. It reassigns the variable to its' existing value, but also divides left-hand side variable value by right-hand side value:

```
<?php
	$ado = 50;

	$ado /= 10;
	echo ado;   // Output 5
?>
```


## Comparison operator

### Non-strict equality comparison operator ( `==` )

This comparison operator is used to compare if the two values are the same.  **The comparison is not strict, meaning it compares only values, and not also the types. Because of that, it is advised not to use this operator - or at least use it with caution.** Comparison operators are used mostly inside the `if` statement condition.

```
<?php
	$a = 10;
	$b = 444;
	$c = 10;
	$d = "10";

	if($a == $b) { echo "The values are the same"; }   // No output.
	if($a == $c) { echo "The values are the same"; }   // "The values are the same"
	if($a == $d) { echo "The values are the same"; }   // "The values are the same"
?>
```

### Non-strict non-equal comparison operator ( `!=` )

This comparison operator is used to compare if the two values ARE NOT the same.  **The comparison is not strict, meaning it compares only values, and not also the types. Because of that, it is advised not to use this operator - or at least use it with caution.** Comparison operators are used mostly inside the `if` statement condition.

```
<?php
	$a = 10;
	$b = 444;
	$c = 10;
	$d = "10";

	if($a != $b) { echo "The values ARE NOT the same"; }   // "The values ARE NOT the same"
	if($a != $c) { echo "The values are the same"; }   // No output.
	if($a != $d) { echo "The values are the same"; }   // No output.
?>
```

### Strict comparison operator ( `===` )

This comparison operator is used to compare if the two values are the same. It not only checks values, but also the types of them. Comparison operators are used mostly inside the `if` statement condition.

```
<?php
	$a = 10;
	$b = 444;
	$c = 10;
	$d = "10";

	if($a == $b) { echo "The values are the same"; }   // No output.
	if($a == $c) { echo "The values are the same"; }   // "The values are the same";
	if($a == $d) { echo "The values are the same"; }   // No output.
?>
```

### Strict non-equal comparison operator ( `!==` )

This comparison operator is used to compare if the two values ARE NOT the same. It not only checks values, but also the types of them. Comparison operators are used mostly inside the `if` statement condition.

```
<?php
	$a = 10;
	$b = 444;
	$c = 10;
	$d = "10";

	if($a !== $b) { echo "The values ARE NOT the same"; }   // "The values ARE NOT the same"
	if($a !== $c) { echo "The values ARE NOT the same"; }   // No output.
	if($a !== $d) { echo "The values ARE NOT the same"; }   // "The values ARE NOT the same"
?>
```

## Logical operators

### OR operator ( `||` / `or` )

Logical operators can be very useful when working with more complex `if` statement conditions. `OR operator` allows to include additional condition part to be verified. If one of the condition parts (or both) are true, then the whole `if` statement will become `true`. Otherwise, if none of the condition parts are true, the `if` statement becomes `false` value. 

In order to express OR operator, you can use either `||` or the literal `or`.

```
<?php
	$first_name = "Tom";
	$last_name = "Smith";

	// At least one condition is satisfied - will output "If statement resolved as true"
	if($first_name === "Anna" || $last_name === "Smith") { 
		echo "If statement resolved as true";
	}

	// None of the conditions are satisfied - no output.
	if($first_name === "John" || $last_name === "Doe") {
		echo "If statement resolved true";
	}
?>
```

### AND operator (`&&` / `and`)

Logical operators can be very useful when working with more complex `if` statement conditions. `AND operator` allows to include additional condition part to be verified. Only if both condition parts are true, then the `if` statement will become  `true`. Otherwise, if none (or simply not all) of the conditions are true, the `if` statement becomes `false` value. 

In order to express OR operator, you can use either `&&` or the literal `and`.

```
<?php
	$age = 25;

	// Both conditions are satisfied - will output "If statement resolved as true"
	if($age >= 18  && $age <= 30) { 
		echo "If statement resolved as true";
	}

	// One or more conditions not satisfied - no output.
	if($age > 20 && age !== 25) {
		echo "If statement resolved true";
	}
?>
```

### Negation operator ( ! )

Logical operators can be very useful when working with more complex `if` statement conditions.
Negation operator `!` placed before a statement simply changes the final value of a given condition. Normally truthy coniditon will be treated as falsy - and vice versa.

```
<?php
	$age = 25;
	$name = "John";

	// Checking if age is NOT less than 18 - will output "If statement resolved as true"
	if(!($age < 18)) { 
		echo "If statement resolved as true";
	}

	// Checking if name is NOT unequal to "Chris" - no output.
	if(!($name !== "Chris")) {
		echo "If statement resolved true";
	}
?>

```


## Incrementing / Decrementing operators

### Incrementing operator ( `++var`, `var++` )

Incrementing operator is widely used for a counter inside the `for` loop. The incrementing operator is a convenient shorthand for `var + 1`.  There are two kinds of incrementing patterns in PHP: 
	1.  *preceding incrementation* - the variable value is first incremented, and then used afterwards;
	2.  *following incrementation* - the variable value is first used, and then incremented;

The difference between the two patterns can be seen in the below code example:

```
<?php
	// Preceding incrementation example
	$pi = 1;
	echo ++$pi;   // Output: 2;
	echo $pi;     // Output: 2;

	// Following incrementation example
	$fi = 1;
	echo $fi++;   // Output: 1;
	echo $fi;     // Output: 2;
?>
```

### Decrementing operator ( `--var`, `var--`)

Decrementing operator is widely used for a counter inside the `for` loop. The decrementing operator is a convenient shorthand for `var - 1`.  There are two kinds of decrementing patterns in PHP: 
	1.  *preceding decrementation* - the variable value is first decremented, and then used afterwards;
	2.  *following decrementation* - the variable value is first used, and then decremented;

The difference between the two patterns can be seen in the below code example:

```
<?php
	// Preceding decrementation example
	$pd = 1;
	echo --$pd;   // Output: 0;
	echo $pd;     // Output: 0;

	// Following decrementation example
	$fd = 1;
	echo $fd--;   // Output: 1;
	echo $fd;     // Output: 0;
?>
```
