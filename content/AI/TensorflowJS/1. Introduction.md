

## What is TensorflowJS ?

TensorflowJS is a Javascript version of a popular Tensorflow library, originally written in Python. The library is used for **machine learning** purposes. TensorflowJS was developed by Google back in 2018 and nowadays it is a great tool when it comes to using AI directly inside the websites or web appications.

## Definitions 

Below are some important definisions and terms widely used in AI industry:

*  **AI** - The science of making things smart. In other words, it is human intelligence exhibited by machines,

* **Narrow AI** - A system that can do just one (or a few) defined things as well or better than humans. That can be : recognizing objects or classifying text.
  
* **Machine Learning (ML)** - An approach to achieve AI through systems that can learn from experience to find patterns in a set of data. Such AI can be then reused and trained with new data without changing the code itself.

* **Deep learning (DL)** - A technique for implementing Machine Learning. It is basically one of many other algorithms to make machine learning program work.
